# positioning-stronghold
### 介绍
这是一个预测Minecraft要塞位置的计算器，采用三角计算法，参考[Minecraft Wiki](https://zh.minecraft.wiki/w/Tutorial:%E7%94%A8%E6%9B%B4%E5%B0%91%E7%9A%84%E6%9C%AB%E5%BD%B1%E4%B9%8B%E7%9C%BC%E5%AE%9A%E4%BD%8D%E8%A6%81%E5%A1%9E "Minecraft Wiki")。

<img width="440" height="160" alt="image" src="https://github.com/user-attachments/assets/672f58cb-5706-46c9-9e2f-62d9312e074a" />

### 教程
在打开程序后，在游戏内投掷一个末影之眼，把游戏**视野**拉到最低，**鼠标灵敏度**调到最低，把准心对准末影之眼中心左侧的像素，可以使用 **F3+Esc** 暂停游戏来保留末影之眼在空中的位置。像这样对齐<img width="60" height="60" alt="image" src="https://github.com/user-attachments/assets/4f554d96-77f3-4cc6-ae70-65b61ce2a24d" />，然后按下 **F3+C** 复制坐标，这样就记录好一个测量点了，然后随便再走10格左右，再重复一次上面的操作，即可计算出坐标。
